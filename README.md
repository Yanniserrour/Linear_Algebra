# Algèbre linéaire

Ce document présente les principales approches numériques utilisées pour la résolution de systèmes d'équations linéaires du type $Ax = b$, classées en deux grandes catégories : les méthodes de résolution directes et les méthodes de résolution indirectes.

## Méthodes de résolution directes

* Méthode de la matrice inverse
* Méthode de Cramer
* Méthodes de triangulation
    * Gauss sans échange
    * Gauss avec échange
    * Gauss pivot partiel
    * Gauss pivot total
* Diagonalisation à travers : Gauss-Jordan
* Méthode de factorisation LU

---

## Méthodes de résolution indirectes

* Méthode de Jacobi
* Méthode de Gauss-Seidel
